# Plataforma ONG — Entrega 1 (HTML5)

Páginas: index.html, projetos.html, cadastro.html  
Assets: assets/css/style.css, assets/js/script.js, assets/img/*

Como abrir:
- Abrir index.html no navegador ou usar Live Server no VS Code.

Observações:
- Formulário com máscaras (CPF/CEP/telefone) e selects de Estado/Cidade.
- Validado com W3C e testado em desktop/mobile.

Este projeto consiste em um site interativo para uma ONG, permitindo que os usuários possam se cadastrar como voluntários e se informar sobre os projetos sociais em andamento. A plataforma foi construída utilizando HTML5, CSS e JavaScript, com validação de campos e funcionalidades como máscaras de CPF, CEP e telefone, além de campos de seleção de estado e cidade.

Funcionalidades Implementadas:
Página Inicial (index.html): Apresenta o logo da ONG, informações sobre os projetos sociais e como o usuário pode ajudar.
Página de Projetos (projetos.html): Detalha os projetos sociais da ONG, como "Projeto de Sementes do Amanhã" e "Projeto Mãos Amigas", com uma breve descrição sobre cada um.
Página de Cadastro (cadastro.html): Permite que os usuários se registrem como voluntários. O formulário possui campos obrigatórios, como nome completo, CPF, telefone, data de nascimento, endereço (CEP, estado, cidade e rua), e validação através de máscaras para CPF, CEP e telefone.

Características Técnicas:
Máscaras de Formulário: Implementação de máscaras de entrada para CPF, telefone e CEP, com o formato adequado para cada tipo de dado.
Campos de Seleção: Implementação de selects dinâmicos para selecionar o estado e a cidade.
Responsividade: O site foi projetado para ser responsivo, funcionando bem tanto em dispositivos desktop quanto móveis.
Validação W3C: O código foi validado conforme as normas do W3C, garantindo a conformidade com os padrões web.

Tecnologias Utilizadas:
HTML5 para estruturação das páginas.
CSS3 para estilização e layout.
JavaScript para interatividade, como as máscaras de entrada e validação dos dados.

Este projeto foi desenvolvido como parte de uma entrega de aprendizagem para uma ONG fictícia, visando proporcionar uma maneira simples e eficaz de novos voluntários se registrarem e se envolverem com a causa da organização.
